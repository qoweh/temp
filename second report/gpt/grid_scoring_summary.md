# build_uiwang_grid_candidates.py 요약

## 이 파일이 뭐야?
- 의왕시 무인교통단속카메라 후보지를 자동으로 뽑는 점수 계산 스크립트입니다.
- 입력 데이터(카메라, 버스정류소, 사고다발지, 사망사고, 학교)를 읽어서 격자별 점수를 만들고, QGIS용 CSV/GeoJSON 파일을 생성합니다.

## 결과는 어떻게 계산했어?
1. 좌표가 있는 점 데이터만 정리해서 불러옵니다.
2. 의왕시 경계를 가져옵니다.
- 기본: OSM 경계 조회
- 실패 시: 점 데이터 외곽(Convex Hull) + 1200m 버퍼로 대체
3. 의왕시 경계 주변 점만 남깁니다.
4. 미터 단위 좌표계(EPSG:5179)로 바꾼 뒤 250m 격자를 만듭니다.
5. 각 격자 중심점에서 최근접 거리를 계산합니다.
- 카메라까지 거리
- 버스정류소까지 거리
- 사고다발지까지 거리
- 사망사고까지 거리
- 학교까지 거리
6. 거리를 0~1 점수로 바꿉니다(10~90 백분위 기준 정규화).
- 카메라: 멀수록 점수 높음
- 버스/사고/사망/학교: 가까울수록 점수 높음
7. 취약도 점수를 만듭니다.
- vulnerability = 0.35(사고) + 0.30(사망) + 0.20(버스) + 0.15(학교)
8. 최종 점수를 계산합니다.
- total = 0.60(카메라와의 거리 점수) + 0.40(취약도)
- 단, 카메라 150m 이내면 최대 0.25 감점
9. 후보지 여부를 판정합니다.
- 카메라와 거리 >= 200m
- 취약도: 55백분위 이상
- 최종점수: 70백분위 이상

## 출력 파일 의미
- uiwang_grid_score_250m.geojson: 전체 격자 점수 지도
- uiwang_candidate_top120_250m.geojson: 우선 검토 후보 중심점
- uiwang_grid_summary.json: 입력 건수/출력 건수/설정값 요약

## 한 줄 해석
